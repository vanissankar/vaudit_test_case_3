# Universal Bank Statement Extractor: Core Libraries

1. PyMuPDF (fitz)
   - **Primary Extraction Engine**: Provides ultra-fast, coordinate-accurate word extraction (`get_text_words`). 
   - Fundamental to the layout-agnostic design, allowing the engine to "see" the document structure spatially.

2. Python 're' (Standard Library)
   - **Intelligence Layer**: Powers the Hybrid Metadata Engine and the Universal Phaser.
   - Handles complex pattern matching for Dates, IFSC codes, and KVP (Key-Value Pair) discovery.

3. pandas
   - **Data Orchestration**: Acts as the centralized container for transaction data.
   - Manages data cleaning, sorting by date, and prepares structured DataFrames for multi-format export.

4. XlsxWriter
   - **Professional Reporting**: High-performance Excel engine used to generate production-ready statements.
   - Supports advanced formatting, including metadata headers and dynamic column auto-sizing.

5. concurrent.futures (Standard Library)
   - **Performance Optimization**: Enables multi-threaded processing (`ThreadPoolExecutor`) in the `Threads/` implementation for rapid batch processing of large statement sets.
