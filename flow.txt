# Universal Bank Statement Extractor: Program Flow

1. [Coordinate-Level Ingestion]
   - **Action**: Load PDF and extract raw words with precise (x0, y0, x1, y1) coordinates.
   - **Tech**: PyMuPDF (fitz)
   - **Result**: Spatial word map for the entire page.

2. [Hybrid Metadata Discovery]
   - **Action**: Scan the first page using a dual-strategy approach:
     - *Strategy A*: Coordinate-aware keyword search for standard banks (SBI/KVB).
     - *Strategy B*: Colon-based Key-Value Pair (KVP) discovery for unknown layouts.
   - **Tech**: engine/detector.py (re)
   - **Result**: Verified Metadata (Bank, Holder, Account No, IFSC, etc.).

3. [Universal Header Detection]
   - **Action**: Dynamically identify the transaction table header by scanning for sets of keywords (Date, Desc, Debit, Credit). Supports multi-line headers (e.g., KVB).
   - **Tech**: engine/parser.py
   - **Result**: Dynamic column boundaries (Midpoint-based zones).

4. [Spatial Transaction Parsing]
   - **Action**: Group words into rows and assign them to semantic roles (Date, Description, Amount) based on their X-coordinate midpoint relative to the dynamic header zones.
   - **Tech**: engine/parser.py
   - **Result**: Raw transaction list.

5. [Amount Isolation & Cleaning]
   - **Action**: Surgically extract amounts from the primary transaction line while ignoring reference numbers or noise in continuation lines to prevent summary overflow.
   - **Tech**: engine/parser.py
   - **Result**: Cleaned financial data.

6. [Robust Aggregation & Summary]
   - **Action**: Consolidated multi-line descriptions, sort by date, and calculate totals. Filters out "Balance Forward" rows to ensure 100% summary accuracy.
   - **Tech**: pandas / Sequential/main.py
   - **Result**: Finalized dataset + Accurate Balance Summary.

7. [Multi-Format Export]
   - **Action**: Export perfectly formatted Excel reports and comprehensive JSON logs.
   - **Tech**: XlsxWriter / json
   - **Result**: output/excel/ and output/json/
